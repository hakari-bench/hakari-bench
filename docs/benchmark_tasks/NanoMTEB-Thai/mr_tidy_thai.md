## Overview

`NanoMTEB-Thai / mr_tidy_thai` is the Thai split of Mr. TyDi retrieval. Thai
questions retrieve Thai Wikipedia-style passages. It evaluates monolingual Thai
dense retrieval for factual questions.

## Details

### What the Original Data Measures

[Mr. TyDi](https://arxiv.org/abs/2108.08787) introduces a multilingual benchmark
for dense retrieval in eleven languages, built from TyDi QA. It is designed for
monolingual retrieval, where each language's queries retrieve documents in the
same language.

The Thai split therefore measures same-language retrieval of relevant evidence
passages, not cross-lingual transfer.

### Observed Data Profile

The Nano split has 200 queries, 10,000 documents, and 234 positive qrel rows.
There are 32 multi-positive queries. Queries average 41.59 characters and
documents average 416.31 characters. The examples ask about provinces,
terrorist attacks, TV shows, country capitals, and greenhouse gases.

Documents are Thai passages with entity titles and explanatory text. Many
queries contain exact names or answer types, which gives sparse retrieval useful
anchors.

### BM25 Difficulty

Using the dataset-provided BM25 candidate column, BM25 reaches nDCG@10 = 0.7291
and hit@10 = 0.8000. It ranks 134 positives at rank 1 and finds a positive in
the top 10 for 160 queries. All positives are within the top 100.

BM25 is strong on this Nano split, but failures show topic confusion. For a
question about Scotland's capital, BM25 can rank a New Zealand country passage
above the Scottish passage because both are country-description pages.

### Training Data That May Help

Thai Mr. TyDi training data, Thai Wikipedia QA retrieval, MIRACL Thai data, and
hard negatives from similar country/person/event pages should help. Avoid
overlap with Mr. TyDi Thai evaluation examples.

### Synthetic Data Guidance

Generate Thai factual questions from Thai passages. Include province counts,
event perpetrators, release dates, capitals, and definition questions. Add
same-category hard negatives, especially other countries, biographies, and
events.
