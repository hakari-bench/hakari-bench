## Overview

`NanoIndicQA / te` is the Telugu split of IndicQA retrieval. Telugu questions
retrieve Telugu context paragraphs.

## Details

### What the Original Data Measures

IndicQA, introduced as part of IndicXTREME in
[Towards Leaving No Indic Language Behind](https://arxiv.org/abs/2212.05409), is
a manually curated cloze-style reading-comprehension task. The MTEB version
turns it into question-to-context retrieval.

### Observed Data Profile

The Nano split has 200 queries, 250 documents, and 200 positive qrel rows. Each
query has one positive. Queries average 64.96 characters and documents average
2,936.18 characters, the longest average document length in NanoIndicQA.

### BM25 Difficulty

Using the dataset-provided BM25 candidate column, BM25 reaches nDCG@10 = 0.5986
and hit@10 = 0.7200. It ranks 96 positives at rank 1 and 144 in the top 10. All
positives are within the top 100.

BM25 benefits from repeated Telugu names and dates, but long documents still
make exact context ranking nontrivial.

### Training Data That May Help

Telugu QA context retrieval, Telugu Wikipedia passage retrieval, and multilingual
IndicQA training are useful. Hard negatives should include other long biography
or literature paragraphs.

### Synthetic Data Guidance

Generate Telugu questions from long context paragraphs. Include biography,
literature, date, place, and title questions, and use paragraph-level positives
with same-topic hard negatives.
