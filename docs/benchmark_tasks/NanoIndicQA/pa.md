## Overview

`NanoIndicQA / pa` is the Punjabi split of IndicQA retrieval. Punjabi questions
retrieve Punjabi evidence paragraphs.

## Details

### What the Original Data Measures

IndicQA is part of IndicXTREME from [Towards Leaving No Indic Language
Behind](https://arxiv.org/abs/2212.05409). It is a cloze-style
reading-comprehension task repurposed by MTEB as question-to-context retrieval.

### Observed Data Profile

The Nano split has 200 queries, 241 documents, and 200 positive qrel rows. Each
query has one positive. Queries average 63.51 characters and documents average
1,423.51 characters. The examples show several Punjabi questions tied to a
single film/actor context.

### BM25 Difficulty

Using the dataset-provided BM25 candidate column, BM25 reaches nDCG@10 = 0.5983
and hit@10 = 0.7600. It ranks 89 positives at rank 1 and 152 in the top 10. All
positives are within the top 100.

### Training Data That May Help

Punjabi QA context retrieval, Punjabi Wikipedia passage retrieval, and
cross-lingual Indic QA training should help. Include hard negatives from similar
entertainment, history, and biography contexts.

### Synthetic Data Guidance

Generate Punjabi questions from paragraphs and keep the full paragraph as the
positive. Use several questions per passage and include negatives with shared
names or entities.
