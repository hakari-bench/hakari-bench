## Overview

`NanoBrightEarthScienceLong` is the long-document version of the Earth Science
StackExchange BRIGHT task. Queries are technical Earth-science posts, and the
retrieval pool contains long cited web pages rather than passage chunks.

## Details

### What the Original Data Measures

[BRIGHT](https://arxiv.org/abs/2407.12883) reports that its long-context
variants convert StackExchange tasks to retrieval from unsplit web pages. This
is meant to test whether systems can find the right source document when the
evidence is buried in long pages, reports, or documentation. For Earth Science,
the source pages cover topics such as geology, climate datasets, planetary
science, and environmental chemistry.

### Observed Data Profile

The split has 116 queries, 587 documents, and 186 positive qrels. Queries
average 476.71 characters, matching the short Earth Science split. Documents
average 70,649.63 characters, and sampled positives include long NASA pages,
Wikipedia pages, geological articles, and mining or science reports. Positives
average 1.60 per query with a median of 1.

### BM25 Difficulty

Using the dataset-provided BM25 candidate column, BM25 reaches nDCG@10 =
0.3282 and hit@10 = 0.5345. It ranks 29 queries with a positive first, and the
median best positive rank is 8.5. Long web pages often include navigation,
boilerplate, and broad topical text, so term overlap can be diluted even when
the document contains the needed evidence.

### Training Data That May Help

Useful data includes long Earth-science web pages aligned with technical
questions, scientific report retrieval, climate-data documentation search, and
document-level supervision derived from cited source pages.

### Synthetic Data Guidance

Generate long source-style pages with sections, tables, caveats, and domain
terminology, then write questions requiring one section's evidence. Hard
negatives should be long pages on the same Earth-science topic that do not
answer the specific query.
