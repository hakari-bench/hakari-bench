## Overview

MS MARCO is a web passage retrieval benchmark. `NanoBEIR-ja__NanoMSMARCO` uses
Japanese translated web-search questions to retrieve Japanese translated
answer-bearing passages.

## Details

### What the Original Data Measures

[MS MARCO](https://arxiv.org/abs/1611.09268) introduced large-scale real user
queries with answer-bearing passages. BEIR includes it as a passage retrieval
task, and MMTEB provides the multilingual evaluation context for the Japanese
translation.

### Observed Data Profile

The sampled task has 50 queries, 5,043 documents, and 50 positive qrels. Every
query has exactly one positive. Queries are short web questions averaging 26.66
characters; documents average 150.35 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.3318 and hit@10 = 0.4600. The median first-positive
rank is 11.5, so lexical matching is often insufficient for Japanese translated
web QA.

### Training Data That May Help

Useful data includes non-overlapping web QA retrieval, Japanese search query
logs, multilingual passage retrieval, and answer-bearing passage pairs. Exclude
MS MARCO, BEIR, NanoBEIR, and overlapping translations.

### Synthetic Data Guidance

Generate concise Japanese web queries from non-evaluation passages. The
positive passage should directly answer the query, while hard negatives should
match surface terms without answering it.
