## Overview

SciFact is a scientific claim evidence retrieval task. `NanoBEIR-pt__NanoSciFact`
uses Portuguese translated scientific claims to retrieve Portuguese translated
abstracts that provide evidence.

## Details

### What the Original Data Measures

[SciFact](https://arxiv.org/abs/2004.14974) evaluates scientific claim
verification using abstracts as evidence. BEIR uses the retrieval portion of the
task, and MMTEB supplies the multilingual context for this Portuguese split.

### Observed Data Profile

The sampled task has 50 queries, 2,919 documents, and 56 positive qrels. Most
queries have one positive, with a small multi-positive tail. Queries average
105.86 characters, and documents average 1,562.73 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.6827 and hit@10 = 0.8400. The median first-positive
rank is 1.0, so scientific terminology gives BM25 strong anchors, but the task
still requires distinguishing evidence from merely related biomedical abstracts.

### Training Data That May Help

Useful data includes non-overlapping scientific claim verification, biomedical
abstract retrieval, Portuguese scientific QA, and multilingual evidence
retrieval. Exclude SciFact, BEIR, NanoBEIR, and translated claim-evidence pairs
from the evaluation split.

### Synthetic Data Guidance

Generate Portuguese scientific claims from abstracts and label the abstract that
supports or contradicts each claim. Hard negatives should share entities,
interventions, or methods while failing to provide the needed evidence.
