## Overview

QuoraRetrieval is a duplicate-question retrieval task. `NanoBEIR-sr__NanoQuoraRetrieval`
uses Serbian translated questions to retrieve Serbian translated duplicate or
near-duplicate questions.

## Details

### What the Original Data Measures

The source is the [Quora Question Pairs](https://kaggle.com/competitions/quora-question-pairs)
duplicate-question dataset. BEIR adapts it as retrieval, and MMTEB provides the
multilingual context.

### Observed Data Profile

The sampled task has 50 queries, 5,046 documents, and 70 positive qrels. Ten
queries have multiple duplicates. Queries average 49.26 characters, and
documents average 58.05 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.5964 and hit@10 = 0.7800. The median first-positive
rank is 1.0, reflecting lexical overlap in many duplicates, with remaining
errors coming from paraphrase variation.

### Training Data That May Help

Useful data includes non-overlapping duplicate-question retrieval, Serbian
paraphrase data, and multilingual semantic question matching. Exclude Quora
Question Pairs, BEIR, NanoBEIR, and translated duplicate records from this split.

### Synthetic Data Guidance

Generate Serbian paraphrase clusters for the same intent. Include hard negatives
that share entities or wording but ask a different question.
