## Overview

ArguAna is an argument-counterargument retrieval benchmark. `NanoBEIR-it__NanoArguAna`
uses Italian translated argumentative passages as queries and retrieves Italian
translated counterarguments or closely paired arguments.

## Details

### What the Original Data Measures

[ArguAna](https://aclanthology.org/P18-1023/) was introduced for argument
retrieval and matching in debate-style text. BEIR includes ArguAna as an
argument retrieval task, and MMTEB provides the multilingual benchmark context.

### Observed Data Profile

The sampled task has 50 queries, 3,635 documents, and 50 positive qrels. Every
query has exactly one positive. Queries are long argumentative passages
averaging 1,187.42 characters, and documents average 1,102.28 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.3934 and hit@10 = 0.7000. The median first-positive
rank is 6. Long text gives BM25 many lexical anchors, but matching the correct
counterargument requires stance and argument-structure understanding.

### Training Data That May Help

Useful data includes non-overlapping argument retrieval, debate counterargument
pairs, stance-aware retrieval, and Italian or multilingual argument mining data.
Exclude ArguAna, BEIR, NanoBEIR, and translated argument records likely to
overlap.

### Synthetic Data Guidance

Generate Italian claims and counterarguments from non-evaluation debate text.
Hard negatives should address the same topic while taking a different stance or
responding to a different premise.
