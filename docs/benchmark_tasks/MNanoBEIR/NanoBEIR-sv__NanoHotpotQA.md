## Overview

HotpotQA is a multi-hop Wikipedia question answering retrieval task.
`NanoBEIR-sv__NanoHotpotQA` uses Swedish translated questions to retrieve
Swedish translated supporting passages.

## Details

### What the Original Data Measures

[HotpotQA](https://arxiv.org/abs/1809.09600) measures retrieval for explainable
multi-hop QA. BEIR evaluates supporting-passage retrieval, and MMTEB provides
the multilingual context.

### Observed Data Profile

The task has 50 queries, 5,090 documents, and 100 positive qrels. Every query
has two positives. Queries average 86.34 characters, and documents average
349.52 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.7413 and hit@10 = 0.9800. The median first-positive
rank is 1.0, but strong performance still requires recovering both supporting
passages.

### Training Data That May Help

Useful data includes non-overlapping multi-hop QA, Swedish Wikipedia retrieval,
and multi-positive passage ranking. Exclude HotpotQA, BEIR, NanoBEIR, and
translated evaluation questions.

### Synthetic Data Guidance

Generate Swedish bridge questions requiring two passages. Hard negatives should
include one mentioned entity but omit the bridging fact.
