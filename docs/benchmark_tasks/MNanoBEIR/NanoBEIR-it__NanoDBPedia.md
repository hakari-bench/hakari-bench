## Overview

DBpedia-Entity is an entity retrieval benchmark. `NanoBEIR-it__NanoDBPedia`
uses Italian translated entity-style queries to retrieve Italian translated
DBpedia passages or entity descriptions.

## Details

### What the Original Data Measures

[DBpedia-Entity V2](https://doi.org/10.1145/3077136.3080751) evaluates entity
search over DBpedia using information needs that may name an entity, describe a
class of entities, or ask for examples. [BEIR](https://arxiv.org/abs/2104.08663)
includes DBpedia-Entity as an entity retrieval task, and
[MMTEB](https://arxiv.org/abs/2502.13595) supplies the multilingual context.

### Observed Data Profile

The sampled Italian Nano task has 50 queries, 6,045 documents, and 1,158
positive qrel rows. It is strongly multi-positive, averaging 23.16 positives per
query. Queries are short entity needs, and documents are compact DBpedia-style
descriptions averaging 361.74 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.5339 and hit@10 = 0.9400 from the dataset BM25 column.
Many positives are easy when names repeat exactly, but broader category queries
still require ranking the best entity descriptions among many valid positives.

### Training Data That May Help

Useful supervision includes non-overlapping entity search, Wikipedia/DBpedia
entity linking, multilingual entity retrieval, and short-query passage
retrieval data. Exclude DBpedia-Entity, BEIR, NanoBEIR, and translated entity
records likely to overlap.

### Synthetic Data Guidance

Generate Italian entity needs from non-evaluation entity descriptions. Include
both exact-name and category-style queries, with hard negatives from related
entities.
