## Overview

MS MARCO is a web passage retrieval benchmark. `NanoBEIR-sr__NanoMSMARCO` uses
Serbian translated web-search questions to retrieve Serbian translated
answer-bearing passages.

## Details

### What the Original Data Measures

[MS MARCO](https://arxiv.org/abs/1611.09268) introduced large-scale real user
queries with answer-bearing passages. BEIR includes it as passage retrieval, and
MMTEB gives the multilingual context for the Serbian translation.

### Observed Data Profile

The sampled task has 50 queries, 5,043 documents, and 50 positive qrels. Every
query has one positive. Queries average 35.64 characters; documents average
331.07 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.2833 and hit@10 = 0.4400. The median first-positive
rank is 12.0, so Serbian translated web QA often requires semantic matching
beyond surface terms.

### Training Data That May Help

Useful data includes non-overlapping web QA retrieval, Serbian search query
logs, multilingual passage retrieval, and answer-bearing passage pairs. Exclude
MS MARCO, BEIR, NanoBEIR, and overlapping translations.

### Synthetic Data Guidance

Generate concise Serbian web queries from non-evaluation passages. The positive
passage should directly answer the query; hard negatives should match surface
terms without answering it.
