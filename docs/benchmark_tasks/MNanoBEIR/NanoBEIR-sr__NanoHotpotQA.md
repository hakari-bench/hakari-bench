## Overview

HotpotQA is a multi-hop Wikipedia question answering retrieval task.
`NanoBEIR-sr__NanoHotpotQA` uses Serbian translated questions to retrieve
Serbian translated supporting passages.

## Details

### What the Original Data Measures

[HotpotQA](https://arxiv.org/abs/1809.09600) was designed for explainable
multi-hop question answering with supporting facts. BEIR evaluates retrieval of
the supporting passages, and MMTEB provides the multilingual context.

### Observed Data Profile

The sampled task has 50 queries, 5,090 documents, and 100 positive qrels. Every
query has two positives. Queries average 86.54 characters, and documents average
353.56 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.6327 and hit@10 = 0.9000. The median first-positive
rank is 1.0, but two-hop coverage still requires retrieving both supporting
documents.

### Training Data That May Help

Useful data includes non-overlapping multi-hop QA retrieval, Serbian Wikipedia
question generation, and multi-positive passage ranking. Exclude HotpotQA,
BEIR, NanoBEIR, and translated evaluation questions.

### Synthetic Data Guidance

Generate Serbian multi-hop questions whose answer requires two linked passages.
Hard negatives should mention one entity from the question but omit the bridge
fact or final answer.
