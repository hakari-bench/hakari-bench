## Overview

MS MARCO is a web passage retrieval benchmark. `NanoBEIR-no__NanoMSMARCO` uses
Norwegian translated web-search questions to retrieve Norwegian translated
answer-bearing passages.

## Details

### What the Original Data Measures

[MS MARCO](https://arxiv.org/abs/1611.09268) introduced large-scale real user
queries with answer-bearing passages. BEIR includes it as a passage retrieval
task, and MMTEB provides the multilingual evaluation context for the Norwegian
translation.

### Observed Data Profile

The sampled task has 50 queries, 5,043 documents, and 50 positive qrels. Every
query has exactly one positive. Queries are short web questions averaging 34.98
characters; documents average 331.30 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.3249 and hit@10 = 0.4800. The median first-positive
rank is 13.5, so lexical matching is often insufficient for Norwegian
translated web QA.

### Training Data That May Help

Useful data includes non-overlapping web QA retrieval, Norwegian search query
logs, multilingual passage retrieval, and answer-bearing passage pairs. Exclude
MS MARCO, BEIR, NanoBEIR, and overlapping translations.

### Synthetic Data Guidance

Generate concise Norwegian web queries from non-evaluation passages. The
positive passage should directly answer the query, while hard negatives should
match surface terms without answering it.
