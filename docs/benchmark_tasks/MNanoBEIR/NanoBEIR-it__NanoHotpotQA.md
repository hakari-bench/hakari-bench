## Overview

HotpotQA is a multi-hop question answering benchmark. `NanoBEIR-it__NanoHotpotQA`
uses Italian translated questions to retrieve Italian translated Wikipedia
paragraphs that contain supporting evidence.

## Details

### What the Original Data Measures

[HotpotQA](https://arxiv.org/abs/1809.09600) was designed for explainable
multi-hop question answering with supporting facts. BEIR treats it as evidence
retrieval, and MMTEB provides the multilingual context for this Italian split.

### Observed Data Profile

The sampled task has 50 queries, 5,090 documents, and 100 positive qrels. Every
query has exactly two positives. Queries average 93.38 characters; documents are
short Wikipedia paragraphs averaging 378.28 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.7275 and hit@10 = 0.9800. Named entities make many
queries lexically easy, but multi-hop questions still require retrieving both
supporting pieces rather than only the most obvious entity page.

### Training Data That May Help

Useful training data includes non-overlapping multi-hop QA retrieval pairs,
Wikipedia evidence selection data, and Italian or multilingual question-to-
passage retrieval. Exclude HotpotQA, BEIR, NanoBEIR, and overlapping translated
support paragraphs.

### Synthetic Data Guidance

Generate Italian multi-hop questions from pairs of non-evaluation passages.
Each generated query should require a bridge entity or comparison, with hard
negatives from one-hop partial matches.
