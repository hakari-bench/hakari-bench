## Overview

QuoraRetrieval is duplicate-question retrieval. `NanoBEIR-th__NanoQuoraRetrieval`
uses Thai translated questions to retrieve Thai translated duplicates.

## Details

### What the Original Data Measures

The source is [Quora Question Pairs](https://kaggle.com/competitions/quora-question-pairs).
BEIR adapts it as duplicate-question retrieval, and MMTEB provides multilingual
context.

### Observed Data Profile

The task has 50 queries, 5,046 documents, and 70 qrels. Ten queries have
multiple positives. Queries average 46.94 characters, and documents average
53.67 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.7283 and hit@10 = 0.8200. Median first-positive rank is
1.0, reflecting strong overlap for many duplicates.

### Training Data That May Help

Use Thai paraphrase and duplicate-question retrieval, excluding Quora, BEIR,
NanoBEIR, and translated evaluation pairs.

### Synthetic Data Guidance

Generate Thai paraphrase clusters with hard negatives that share wording but ask
a different question.
