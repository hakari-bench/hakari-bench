## Overview

FiQA is a financial question-answer retrieval task. `NanoBEIR-pt__NanoFiQA2018`
uses Portuguese translated finance questions to retrieve Portuguese translated
answer passages.

## Details

### What the Original Data Measures

[FiQA](https://doi.org/10.1145/3184558.3192301) was introduced for financial
opinion and question answering data. In BEIR, the retrieval task asks models to
rank answer-bearing financial forum passages; MMTEB provides the multilingual
evaluation context for this Portuguese version.

### Observed Data Profile

The sampled task has 50 queries, 4,598 documents, and 123 positive qrels. More
than half of the queries have multiple positives, with an average of 2.46 and a
maximum of 15. Queries average 71.92 characters, while documents average 972.51
characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.2621 and hit@10 = 0.4600. The median first-positive
rank is 13.5, so financial answer retrieval often needs semantic matching beyond
surface overlap.

### Training Data That May Help

Helpful data includes non-overlapping financial QA, Portuguese finance forum
retrieval, and answer ranking with multiple acceptable answers. Exclude FiQA,
BEIR, NanoBEIR, and translated evaluation answers.

### Synthetic Data Guidance

Generate Portuguese personal-finance and investing questions from source-style
answer passages. Hard negatives should use the same financial product or tax
terms but answer a different decision problem.
