## Overview

MS MARCO is a web passage retrieval benchmark. `NanoBEIR-sv__NanoMSMARCO` uses
Swedish translated web-search questions to retrieve answer-bearing passages.

## Details

### What the Original Data Measures

[MS MARCO](https://arxiv.org/abs/1611.09268) introduced large-scale user queries
and answer-bearing passages. BEIR includes it as passage retrieval, and MMTEB
adds the multilingual evaluation context.

### Observed Data Profile

The task has 50 queries, 5,043 documents, and 50 positive qrels. Every query has
one positive. Queries average 33.60 characters, and documents average 321.19
characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.3777 and hit@10 = 0.5400. The median first-positive
rank is 6.5, so lexical retrieval helps but does not solve the Swedish web-QA
matching problem.

### Training Data That May Help

Use non-overlapping Swedish web QA, search-query logs, multilingual passage
retrieval, and answer-bearing passage pairs. Exclude MS MARCO, BEIR, NanoBEIR,
and translated overlaps.

### Synthetic Data Guidance

Generate short Swedish web questions from non-evaluation passages. Hard
negatives should share surface terms without answering the question.
