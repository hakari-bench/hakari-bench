## Overview

MS MARCO is a web passage retrieval benchmark. `NanoBEIR-it__NanoMSMARCO` uses
Italian translated web-search questions to retrieve Italian translated passages.

## Details

### What the Original Data Measures

[MS MARCO](https://arxiv.org/abs/1611.09268) introduced large-scale real user
queries with answer-bearing passages. BEIR includes it as a passage retrieval
task, and MMTEB provides the multilingual evaluation context for the Italian
translation.

### Observed Data Profile

The sampled task has 50 queries, 5,043 documents, and 50 positive qrels. Every
query has exactly one positive. Queries are short web questions averaging 42.04
characters; documents average 356.59 characters.

### BM25 Difficulty

BM25 reaches nDCG@10 = 0.3957 and hit@10 = 0.6000. The median first-positive
rank is 5.5, indicating that lexical retrieval is useful but often misses
answer-bearing paraphrases.

### Training Data That May Help

Useful data includes non-overlapping web QA retrieval, Italian search query
logs, multilingual passage retrieval, and answer-bearing passage pairs. Exclude
MS MARCO, BEIR, NanoBEIR, and overlapping translations.

### Synthetic Data Guidance

Generate concise Italian web queries from non-evaluation passages. The positive
passage should directly answer the query, while hard negatives should match
surface terms without answering it.
