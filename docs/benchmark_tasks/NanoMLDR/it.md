## Overview

`it` is the Italian split of NanoMLDR. It evaluates Italian long-document
retrieval from question-style information needs.

## Details

### What the Original Data Measures

[M3-Embedding](https://arxiv.org/abs/2402.03216) uses MLDR to evaluate
multilingual long-document retrieval with NDCG@10. The [MLDR dataset card](https://huggingface.co/datasets/Shitao/MLDR)
lists Italian as Wikipedia-sourced and describes question generation from a
sampled paragraph while using the full article as the retrieval target.

### Observed Data Profile

The Nano split has 158 queries, 3,116 documents, and 158 positive qrels. Each
query has one positive. Queries average 98.16 characters and documents average
14,374.38 characters. Sample topics include cities, musicians, film awards,
football teams, and macroeconomic history.

### BM25 Difficulty

Using the dataset-provided BM25 candidate column, BM25 reaches nDCG@10 =
0.7635 and hit@10 = 0.8354. It ranks 107 positives first. Lexical matching is
usually strong, but some questions require locating a specific detail deep in a
long biography or topic article.

### Training Data That May Help

Useful training data includes Italian Wikipedia question-article pairs,
Italian long-document QA retrieval, multilingual MLDR training data, and
hard negatives from articles sharing people, places, events, or technical terms.

### Synthetic Data Guidance

Synthetic data should create Italian questions from a target paragraph in a long
article. Negatives should be full Italian articles from adjacent topics that
share named entities but not the answer.
